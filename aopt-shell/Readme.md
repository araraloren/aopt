# aopt-shell

Shell completion support for aopt framework.

## Shell

PROGRAM is the name of binary

### bash

```bash
```

### fish

```fish
```

### Zsh

```zsh
```

### Powershell

```powershell
Add-Content $PROFILE "`nPROGRAM --_shell powershell | Out-String | Invoke-Expression"
```
